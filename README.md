# AI-Based Sentiment Analysis System

## Project Overview
This project uses Machine Learning and Deep Learning techniques to analyze movie reviews and classify them as positive or negative sentiments.

## Technologies Used
- Python
- Pandas
- NumPy
- NLTK
- Scikit-learn
- TensorFlow/Keras
- Matplotlib
- Streamlit

## Algorithms Used
- Naïve Bayes
- Logistic Regression
- LSTM

## Dataset
IMDB Movie Reviews Dataset

## Features
- Text preprocessing
- TF-IDF feature extraction
- ML and DL model training
- Accuracy comparison
- Confusion matrix visualization
- Streamlit deployment

## Results
- Naïve Bayes Accuracy: 85%
- Logistic Regression Accuracy: 89%
- LSTM Accuracy: 92%
